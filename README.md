# Extension_translate_English_Darija
Extension Chrome Manifest V3 permettant la traduction instantanée de l'anglais vers le darija marocain via menu contextuel et panneau latéral. Utilise un service REST JAX-RS sécurisé par authentification basique, connecté à l'API Google Gemini pour la traduction par LLM.
